# 🚀 TrackAssist – AI Chatbot

## ⚡ Quickest Start
Double-click `index.html` → works instantly in any browser. No setup needed.

## 🐍 Run with Python Backend (Optional)
```
pip install -r requirements.txt
python app.py
# Open http://localhost:5000
```

## 🌐 Host Free Online
Drag this folder onto https://netlify.com → live URL in 30 seconds.

## 📁 Files
| File            | Purpose                                      |
|-----------------|----------------------------------------------|
| index.html      | Standalone app — 914 keywords embedded       |
| app.py          | Flask backend server                         |
| kword.xlsx      | 914 keywords → 100 response file mappings    |
| requirements.txt| Python dependencies                          |
| Headache/       | 100 response .txt files                      |
